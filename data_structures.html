Stacks (LIFO: Last-In, First-Out)

Explanation & Conceptual Example:

A Stack is a linear data structure that follows the LIFO (Last-In, First-Out) principle. Imagine a stack of dinner plates; you can only add (Push) a new plate to the top, and you can only remove (Pop) the plate that was last placed on top.

Example: Pushing 'A', then 'B', then 'C'. The Pop order will be 'C', then 'B', then 'A'.

Queues (FIFO: First-In, First-Out)

Explanation & Conceptual Example:

A Queue is a linear data structure that follows the FIFO (First-In, First-Out) principle. Imagine people waiting in a line; the first person in line (First-In) is the first person to be served (First-Out).

Example: Enqueuing 'A', then 'B', then 'C'. The Dequeue order will be 'A', then 'B', then 'C'.

Binary Tree Traversal (Hierarchical Structure)

Explanation & Conceptual Example:

A Tree is a non-linear data structure where data is organized hierarchically and connected by edges. In a Binary Tree, each node has at most two children (left and right). 

// Example Binary Search Tree Structure used for simulation:
// Values: 10, 20, 30, 40, 50, 60
//
//               40 (Root)
//              /   \
//            20      50
//           / \       \
//         10   30      60

Example Tree Structure: A Binary Search Tree containing values **10, 20, 30, 40, 50, 60** with 40 as the root. Traversal methods dictate the order in which these nodes are visited.

  • Inorder: Visits nodes in ascending order (10, 20, 30, 40, 50, 60).
  • **Preorder: Visits Root first (40, 20, 10, 30, 50, 60).
  • Postorder: Visits Root last (10, 30, 20, 60, 50, 40).

Informational Data Structures (Arrays)


                

                    
C++ Arrays 

                    
                    

                        

                            
1. Fixed-Size (Static) Array

                            
Memory is allocated at compile time. Size cannot change during runtime.

                        

                        
// C++ Fixed-Size Array
int fixed_array[5] = {10, 20, 30, 40, 50};

                    

                    
                    

                        

                            
2. Resizable (Dynamic) Array - `std::vector`

                            
An array that can grow or shrink in size at runtime, handled using the standard library container `std::vector`.

                        

                        
// C++ Dynamic Array
#include <vector>
std::vector<int> dynamic_array; 
dynamic_array.push_back(3);

                    

                    
                    

                        

                            
3. One-Dimensional (1D) Array

                            
A linear list of elements stored in contiguous memory locations. Elements are accessed using a single index.

                        

                        
// C++ 1D Array
int grades[4] = {88, 92, 75, 95};

                    

                    
                    

                        

                            
4. Multi-Dimensional (2D) Array / Matrix

                            
An array of arrays, often used to represent grids or tables (matrices). Accessed using row and column indices.

                        

                        
// C++ 2D Array
int matrix[3][2] = {{1, 2}, {3, 4}, {5, 6}};

                    

                


                

                    
Python Arrays (Lists) 

                    
                    

                        

                            
1. Standard List (Dynamic/Resizable)

                            
Python's built-in `list` is inherently dynamic and resizable, ideal for general collections.

                        

                        
# Python Standard List
dynamic_list = [10, 'hello', 3.14]

                    

                    
                    

                        

                            
2. Fixed-Type Array (`array` module)

                            
Used for storing only homogeneous data (all same type) more efficiently than a standard list, similar to a C array.

                        

                        
# Python Fixed-Type Array
from array import array 
fixed_array = array('i', [1, 2, 3])

                    

                    
                    

                        

                            
3. One-Dimensional (1D) Array

                            
A single sequence of elements, which is a standard Python list containing scalar values.

                        

                        
# Python 1D Array
ages = [18, 22, 25, 30]

                    

                    
                    

                        

                            
4. Multi-Dimensional (2D) Array / Matrix

                            
A list of lists, representing rows and columns, commonly used for matrix operations.

                        

                        
# Python 2D Array
matrix = [[1, 2, 3], [4, 5, 6]]
