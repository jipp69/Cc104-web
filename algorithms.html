10 Core Algorithms (Explanation & Interactive Example)

1. Linear Search +

Checks every element in a list sequentially until a match is found or the end of the list is reached. [attachment_0](attachment)

Short Example:

Searching for the number **7** in the array **[2, 5, 7, 9, 15]**. The search starts at index 0, checks 2, 5, and stops at index 2 when **7** is found.

2. Binary Search +

Efficiently locates an item in a **sorted** list by repeatedly dividing the search interval in half.

Short Example:

Searching for **30** in **[10, 20, 30, 40, 50]**. Step 1: Check the middle (30). Found immediately. If searching for 45, it would eliminate the left half (10, 20, 30) and check the middle of the remaining list.

3. Bubble Sort +

A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Short Example:

Sorting **[5, 1, 4]**.
Pass 1: (5, 1) swap $\rightarrow$ [1, 5, 4]. (5, 4) swap $\rightarrow$ **[1, 4, 5]**.

4. Selection Sort +

Finds the minimum element from the unsorted portion of the list and swaps it with the element at the current position.

Short Example:

Sorting **[5, 1, 4]**.
Step 1: Find min (1), swap with 5 $\rightarrow$ **[1, 5, 4]**.
Step 2: Find min from [5, 4] (which is 4), swap with 5 $\rightarrow$ **[1, 4, 5]**.

5. Factorial (Iterative) +

Calculates the product of all positive integers less than or equal to a given positive integer (n!).

Short Example:

Calculate $4!$. The algorithm computes $1 \times 2 \times 3 \times 4$, resulting in **24**.

6. Fibonacci Sequence +

Generates a sequence where each number is the sum of the two preceding ones, usually starting with 0 and 1.

Short Example:

The sequence starts $F(0)=0$, $F(1)=1$.
$F(2) = F(1) + F(0) = 1 + 0 = 1$.
$F(3) = F(2) + F(1) = 1 + 1 = 2$.

7. Euclidean Algorithm (GCD) +

Computes the greatest common divisor (GCD) of two integers using repeated division/subtraction.

Short Example:

Find $\text{GCD}(48, 18)$.
$48 = 2 \times 18 + 12$.
$18 = 1 \times 12 + 6$.
$12 = 2 \times 6 + 0$.
The last non-zero remainder is **6**.

8. Palindrome Check +

Determines if a string or number reads the same forwards and backwards.

Short Example:

Checking "racecar". The algorithm compares 'r' (start) with 'r' (end), 'a' with 'a', 'c' with 'c'. Since all match, it returns **TRUE**.

9. Sum Array Elements +

Computes the total sum of all numerical values present within an array or list by iteration.

Short Example:

Summing **[1, 5, 10, 2]**. The algorithm starts at $0$, then adds $1 \rightarrow 1$, adds $5 \rightarrow 6$, adds $10 \rightarrow 16$, adds $2 \rightarrow 18$. The total sum is **18**.

10. Reverse String +

Creates a new string by iterating through the original string from end to start.

Short Example:

Reversing "HELLO". The algorithm reads 'O', then 'L', then 'L', then 'E', then 'H', resulting in **OLLEH**.
