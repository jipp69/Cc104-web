10 Core Algorithms (Pseudocode & Interactive Explanation)

1. Linear Search Pseudocode +

FUNCTION LinearSearch(Array, Target):
    FOR i FROM 0 TO length(Array) - 1:
        IF Array[i] == Target:
            RETURN i
    RETURN -1
Short Explanation:

This pseudocode uses a simple loop (`FOR`) to iterate through every index of the array until the `Target` element is found, returning its position (`i`). If the loop finishes without a match, it returns `-1`.

2. Binary Search Pseudocode +

FUNCTION BinarySearch(Array, Target):
    Start = 0, End = length(Array) - 1
    WHILE Start <= End:
        Mid = (Start + End) / 2
        IF Array[Mid] == Target:
            RETURN Mid
        ELSE IF Array[Mid] < Target:
            Start = Mid + 1
        ELSE:
            End = Mid - 1
    RETURN -1
Short Explanation:

This code iteratively cuts the search space in half (`WHILE Start <= End`). It checks the `Mid` point and adjusts `Start` or `End` based on whether the `Target` is greater or smaller than the value at `Mid`.

3. Bubble Sort Pseudocode +

FUNCTION BubbleSort(Array):
    n = length(Array)
    FOR i FROM 0 TO n - 2:
        FOR j FROM 0 TO n - i - 2:
            IF Array[j] > Array[j + 1]:
                Swap(Array[j], Array[j + 1])
Short Explanation:

This uses two nested loops. The inner loop compares adjacent elements and swaps them if they are out of order, causing the largest unsorted element to "bubble up" to its correct position in each pass of the outer loop.

4. Selection Sort Pseudocode +

FUNCTION SelectionSort(Array):
    n = length(Array)
    FOR i FROM 0 TO n - 2:
        Min_Index = i
        FOR j FROM i + 1 TO n - 1:
            IF Array[j] < Array[Min_Index]:
                Min_Index = j
        Swap(Array[i], Array[Min_Index])
Short Explanation:

The outer loop tracks the current position (`i`). The inner loop searches for the smallest remaining element (`Min_Index`) in the rest of the array. Once found, it swaps that smallest element with the element at the current position (`Array[i]`).

5. Factorial Pseudocode +

FUNCTION Factorial(n):
    Result = 1
    FOR i FROM 1 TO n:
        Result = Result * i
    RETURN Result
Short Explanation:

This is the iterative approach. It initializes `Result` to $1$ (since $0! = 1$) and then multiplies the `Result` by every integer from 1 up to and including $n$.

6. Fibonacci Pseudocode +

FUNCTION Fibonacci(n):
    IF n <= 1: RETURN n
    RETURN Fibonacci(n - 1) + Fibonacci(n - 2)
Short Explanation:

This is the recursive approach. It defines the base cases for $n=0$ and $n=1$, and then calls itself to calculate the two preceding numbers, summing them to find the $n$-th number.

7. Euclidean Algorithm Pseudocode +

FUNCTION GCD(a, b):
    WHILE b != 0:
        temp = b
        b = a MOD b
        a = temp
    RETURN a
Short Explanation:

This code repeatedly replaces the larger number (`a`) with the smaller number (`b`) and the smaller number (`b`) with the remainder of their division (`a MOD b`). The loop continues until the remainder (`b`) is zero, at which point `a` holds the GCD.

8. Palindrome Check Pseudocode +

FUNCTION IsPalindrome(String):
    Left = 0, Right = length(String) - 1
    WHILE Left < Right:
        IF String[Left] != String[Right]:
            RETURN FALSE
        Left = Left + 1
        Right = Right - 1
    RETURN TRUE
Short Explanation:

It uses two pointers, `Left` (start) and `Right` (end), moving inward (`WHILE Left < Right`). If the characters at these pointers ever do not match, the string is not a palindrome and it immediately returns `FALSE`.

9. Sum Array Elements Pseudocode +

FUNCTION SumArray(Array):
    Total = 0
    FOR element IN Array:
        Total = Total + element
    RETURN Total
Short Explanation:

This is a simple aggregation loop. It initializes a counter (`Total = 0`) and then iterates through every item in the `Array`, adding its value to the `Total` accumulator.

10. Reverse String Pseudocode +

FUNCTION ReverseString(InputString):
    NewString = ""
    FOR i FROM length(InputString) - 1 DOWNTO 0:
        NewString = NewString + InputString[i]
    RETURN NewString
Short Explanation:

The loop starts from the last index (`length(InputString) - 1`) and decrements (`DOWNTO 0`). It concatenates each character to the `NewString` in reverse order, effectively flipping the input.
